# Random Task Generator (Генератор случайных задач)

**Автор:** Пугановский Василий Михайлович

## Описание

Random Task Generator - это GUI-приложение на Python, которое помогает случайным образом выбирать задачи из предопределенного списка. Приложение поддерживает категоризацию задач, сохранение истории и фильтрацию по категориям.

## Функциональность

- Генерация случайных задач из предопределенного списка
- Добавление новых задач с указанием категории
- Фильтрация задач по категориям: учёба, спорт, работа
- Отображение истории сгенерированных задач
- Сохранение истории в JSON-файл
- Загрузка истории при запуске приложения
- Проверка корректности ввода (запрет пустых задач)
